To use the Code simply download the whole repository to your computer and run the R files.

.R Files
CodeAnalysis: contains all the core analysis concerned with Gross domestic product per capita
CodeComparison: contains a brief regression to compare the results using different data for the cultural dimensions

#/datafiles
6-dimensions-for-website-2015-08-16 adapt.xls
  source: https://geerthofstede.com/research-and-vsm/dimension-data-matrix/
  Contains data on cultural dimensions. Is adapted to match country names / codes with the other datafiles
API_NY.GDP.PCAP.CD_DS2_en_excel_v2_4901640.xls
  source: https://data.worldbank.org/indicator/NY.GDP.PCAP.PP.CD
  Contains data on the gross domestic product per capita in international $
SDR2020Database.xlsx
  source: https://www.sdgindex.org/reports/sustainable-development-report-2020/

sdgmapping.xlsx
  Contains SDG Numbers and full names
